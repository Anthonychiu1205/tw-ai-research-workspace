# Scenarios

v0.5 adds guided, deterministic scenario workflows for demo-ready research operations.

Scenario IDs:
- `analyze_2330`
- `generate_2330_report`
- `compare_ai_server_watchlist`
- `inspect_planner_trace`
- `evaluate_phase2_signals`

All scenarios are mock-safe by default and do not require backend connectivity.

Boundaries:
- no dashboard/SaaS buildout
- no trading/order execution
- no broker integration
- no financial advice output intent
