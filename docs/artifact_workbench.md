# Artifact Workbench

v0.5 artifact workbench supports demo inspection and sharing.

Features:
- filterable browser
- metadata + lineage
- JSON viewer/actions
- pin/unpin
- share bundle participation

All artifacts carry mock/synthetic/non-advice/no-trading metadata.

v0.8 adds:
- `portfolio_review`
- `rebalance_plan`
- `backtest_v2_summary`
