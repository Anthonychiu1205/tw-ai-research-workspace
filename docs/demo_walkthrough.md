# Demo Walkthrough

Use this checklist for a local workspace demonstration.

1. Open workspace (`/workspace`).
2. Confirm runtime mode (`mock` or `api`) in topbar/runtime settings.
3. Check backend status card and fallback indicator.
4. Run Analyze 2330 scenario.
5. Generate a research report artifact.
6. Open evidence timeline and inspect evidence IDs.
7. Open planner trace and reflection summary.
8. Run portfolio review for watchlist.
9. Run backtest v2 (portfolio manager mode).
10. Compare strategies and inspect synthetic metrics.
11. Export a workspace share bundle.
12. Explain current limitations and safety boundaries.

## Safety language for demos

- research-oriented synthetic outputs
- hypothetical backtest context
- non-advice disclaimer
- no broker integration
- no trade/order execution
